STOON - FILIERES LIEES AUX ETABLISSEMENTS PAR VILLE
Version de travail pour seed data / Codex

IMPORTANT
- Ce fichier relie chaque établissement de la liste STOON aux filières/domaines principaux à proposer dans les filtres.
- Pour les facultés et grandes écoles publiques, les filières suivent les familles indiquées dans le répertoire AMCI 2024-2025.
- Pour les établissements privés, les filières sont une base fonctionnelle par domaine d'école. Les intitulés exacts accrédités peuvent changer selon l'année universitaire et doivent être vérifiés dans la liste officielle MESRSI des filières accréditées.
- Pour STOON, cette base est suffisante pour créer les menus déroulants Région > Ville > Etablissement > Filière, puis affiner progressivement.
